# 🐍 Snake & Ladders 🐍
> ✨ A fun multiplayer Snake & Ladders game built with pure HTML, CSS & JavaScript — featuring SVG snakes & ladders, smooth token animations, and a soft girl aesthetic. No frameworks, no backend, just clean vanilla JS! 🎀

---

## 🎮 Live Demo
👉 [Click here to play!](https://Pooja-Vo3.github.io/snake-and-ladders)

---

## 🌸 Features
- 🤖 **Single Player** — You vs Computer (CPU auto-plays)
- 👯 **Multiplayer** — 2 to 4 players on the same screen
- 🐍 **SVG Snakes** — Wavy snakes with eyes, drawn dynamically
- 🪜 **SVG Ladders** — Ladders with rails and rungs
- 🧁 **Smooth Token Animation** — Tokens glide across the board
- 🎲 **Dice Roll Animation** — Satisfying dice effect
- 📜 **Live Game Log** — Every move tracked in real time
- 🏆 **Win Screen** — Confetti celebration for the winner!
- 🎀 **Soft Girl Aesthetic** — Pink, lavender & mint palette

## 📁 Project Structure
```
snake-and-ladders/
├── index.html        ← 3 screens: Mode, Setup, Game
├── README.md
├── css/
│   └── style.css     ← All styling & animations
└── js/
    ├── config.js     ← Constants, snakes & ladders positions
    ├── board.js      ← Board grid, SVG drawing, token movement
    ├── game.js       ← Game state, turns, move logic
    ├── ui.js         ← DOM rendering, animations, screen management
    └── main.js       ← App entry point
```

---

## 🚀 How to Run
Simply double click `index.html` — works in any browser, no installation needed!
